## Install (recommended: GitHub + npx)

Push this folder to a GitHub repo (**we suggest keeping the repo name as `aicz-skill`**), then anyone can install with:

```bash
npx skills add personamarket/aicz-skill
```

This drops the skill straight into Claude Code at `~/.claude/skills/aicz-skill/`.

### Push this package to GitHub

```bash
unzip aicz-skill.zip
cd aicz-skill
git init
git add .
git commit -m "Add Changpeng Zhao(CZ) Persona Skill"
gh repo create aicz-skill --public --source=. --remote=origin --push
```

Your README will then render on GitHub, and the install command becomes `npx skills add <your-username>/aicz-skill`.

### Local install (optional)

```bash
unzip aicz-skill.zip
cp -r aicz-skill ~/.claude/skills/
```
